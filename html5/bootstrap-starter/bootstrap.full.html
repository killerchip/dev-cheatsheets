Bootstrap Full starter template

Use this HTML page as a blank template with full bootstrap capabilities. It includes full boostrap css, and all 3 required Javascript files: Jquery, Popper and Bootstrap plugins.

Use this template if you are requiring use of the following components:

  • Dropdowns for displaying and positioning (also requires Popper.js)
  • Tooltips and popovers for displaying and positioning (also requires Popper.js)
